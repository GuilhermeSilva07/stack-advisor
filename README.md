# StackAdvisor

## O que é
Agente de IA que recomenda a stack tecnológica ideal para seu projeto.

## Como funciona
- Executa um quiz de 8 perguntas sobre o projeto.
- Gera uma recomendação personalizada de stack (frontend, backend, banco de dados, deploy, etc.).

## Tecnologias usadas
- Zed Agent
- Markdown
- Firecrawl (em breve)
