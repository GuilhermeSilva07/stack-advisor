# Project Profile

## Resumo do Quiz
- **Tipo de projeto:** Full Stack
- **Descrição:** Registro de atividades físicas de atletas de ciclismo
- **Tamanho da equipe:** Solo
- **Prazo estimado:** 1 a 3 meses
- **Orçamento:** Gratuito
- **Experiência da equipe:** Iniciante
- **App mobile:** Não
- **Requisitos especiais:** Nenhum

## Stack Recomendada
- **Frontend:** Next.js (deploy no Vercel)
- **Backend:** Node.js + Express (deploy no Railway)
- **Banco de Dados:** PostgreSQL (Supabase free tier)
- **Observabilidade:** LogRocket ou Sentry (planos gratuitos)

## Justificativa
- **Simplicidade:** Tecnologias populares com boa documentação, ideais para iniciantes.
- **Custo Zero:** Todas as opções têm camadas gratuitas que cobrem o escopo do projeto.
- **Tempo de entrega:** Stack pronta para desenvolvimento rápido, permitindo MVP em < 3 meses.
